GEOStar APIs - Mobile Authentication

API Host Configuration

Change this if your Django server is running on a different host/port

1. Mobile Login

Endpoint: http://localhost:8000/api/auth/login/
JSON: {"username": "admin", "password": "1234"}
Headers: 
Content-Type: application/json
Body: 
{
  "username": "admin",
  "password": "1234"
}

2. Mobile Register

Endpoint: http://localhost:8000/api/auth/register/
JSON: {"username": "newuser", "email": "newuser@example.com", "password": "1234", "password2": "1234"}
Headers: 
Content-Type: application/json
Body: 
{
  "username": "newuser",
  "email": "newuser@example.com",
  "password": "1234",
  "password2": "1234"
}

3. Google Login

Endpoint: http://localhost:8000/api/auth/google/
JSON: {"access_token": "your_google_access_token_here"}
Headers: 
Content-Type: application/json
Body: 
{
  "access_token": "your_google_access_token_here"
}

4. GitHub Login

Endpoint: http://localhost:8000/api/auth/github/
JSON: {"access_token": "your_github_access_token_here"}
Headers: 
Content-Type: application/json
Body: 
{
  "access_token": "your_github_access_token_here"
}

5. Password Reset Request

Endpoint: http://localhost:8000/api/auth/password/reset/
JSON: {"email": "user@example.com"}
Headers: 
Content-Type: application/json
Body: 
{
  "email": "user@example.com"
}

6. Password Reset Confirm

Endpoint: http://localhost:8000/api/auth/password/reset/confirm/
JSON: {"uid": "MQ", "token": "abc123", "new_password": "newpass123", "new_password2": "newpass123"}
Headers: 
Content-Type: application/json
Body: 
{
  "uid": "MQ",
  "token": "abc123",
  "new_password": "newpass123",
  "new_password2": "newpass123"
}

7. Password Change (Requires Auth)

Authentication Status: No token - Please login first
Current Token: None
Endpoint: http://localhost:8000/api/auth/password/change/
JSON (3 fields): {"old_password": "oldpass123", "new_password": "newpass123", "new_password2": "newpass123"}
Headers: 
Authorization: Bearer [token_will_appear_here]
Content-Type: application/json
Body: 
{
  "old_password": "oldpass123",
  "new_password": "newpass123", 
  "new_password2": "newpass123"
}
